### Installation

```
git clone https://github.com/EduardoP103/Proyect-Django.git
```
```
pip install -r requirements.txt
```
```
python manage.py runserver
```

now you can visit <a href="http://localhost:8000" target="_blank">http://localhost:8000</a>
